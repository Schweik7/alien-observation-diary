# 地球人观察日记 · 在线多人版

实时多人猜测游戏：选一个外星观察员代号 → 创建房间 → 把房间号/链接发给家人朋友 → 一起猜猜彼此。

基于主规则的「合作猜测」机制：每轮一人当**样本**（秘密选出真实反应），其余人当**观察员**（秘密猜样本会怎样），同时翻牌，猜中越多，全家的**理解值**越高。

## 技术栈
- 后端：Node + Express + Socket.IO（房间与实时同步，状态在内存中）
- 前端：Vite + React + socket.io-client
- 牌库：复用项目的三套 CSV（高年级 / 低年级 / 夫妻版），服务端启动时解析

## 运行

```bash
cd game-app
pnpm install
```

**开发模式**（前后端分开，前端热更新）：
```bash
pnpm dev
# 前端 http://localhost:5174 （在浏览器打开这个）
# 后端 socket 服务 http://localhost:3001
```

**生产/局域网模式**（一个端口同时托管前端和后端）：
```bash
pnpm build      # 构建前端到 ../dist
pnpm start      # 访问 http://localhost:3001
```

## 怎么和别人一起玩
- 同一台机器 / 局域网：让别人访问 `http://<你的IP>:3001`（先 `pnpm build && pnpm start`），或开发模式下访问 `http://<你的IP>:5174`。
- 公网邀请：用 `ngrok http 3001`（或 cloudflared）把本机端口暴露成公网链接，把链接发给对方即可。
- 进入后选好代号 → 创建房间 → 顶部「复制邀请链接」发给对方；对方打开链接会自动填好房间号。

## 玩法说明
- 支持 **2–8 人**。每轮样本顺序自动轮换。
- 抽到**事件卡** → 玩法 A：从四色反应（🔴炸毛/🔵讲理/🟢求抱/🟡躲开）里选。
- 抽到**反应卡** → 玩法 B：从配套的 3 张场景卡里选「哪张真的最会让我这样」。
- 计分（与主规则一致，按人数缩放）：
  - 目标理解值 = `5 + 3 ×（观察员人数）`
  - 失败上限（无人猜中累计）= `玩家数 + 1`
  - 每位猜中的观察员 +1；全员猜中再 +1；无人猜中 −1 且计一次失败。
- 🔴 红卡（掏心窝）任何人可在出牌阶段点「跳过」。
- 翻牌后会提示「先好奇、别辩解」——这是这个游戏真正的意义。

> 说明：在线版对所有牌库统一采用上面这套按人数缩放的计分（夫妻版桌面规则里"无第三方/误解值"的变体未在线上实现，线上靠人数缩放即可平衡）。

## 目录结构
```
game-app/
  server/
    index.js     Express + Socket.IO 入口，生产环境托管 dist
    game.js      房间与游戏状态机（轮换、计分、胜负、结算）
    decks.js     CSV 解析 + 把卡编译成"可玩单元"（玩法A/玩法B）
    decks/*.csv  三套牌库副本
  client/
    src/
      App.jsx          socket 订阅 + 路由（首页/观测室/对局/结算）
      net.js           socket 连接
      nicknames.js     预设观察员代号
      screens/         Home / Lobby / GameBoard / GameOver
      styles.css       外星观测主题样式
  vite.config.js  root=client，构建到 ../dist
```
